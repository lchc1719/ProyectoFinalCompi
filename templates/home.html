Asignaciones a variables:
Se dice la variable en la que se va a guardar el valor, seguido del signo igual y del valor a guardar en la variable, los nombres de variables inician por una letra o guión bajo. se puede crear una variable de tipo cadena, numérico o booleano (verdadero/falso), al final se pone punto y coma
Operaciones:
Se pueden operar variables, se pueden hacer sumas '+', restas '-', divición '/' y multiplicación '*'.
Imprimir:
Para imprimir solo se debe escribir la palabra reservada 'imprime' seguido de dos puntos y el valor a imprimir, dicho valor puede ser el de una variable un numero, una cadena o un esstado booleano.
Sentencia si y si no:
Se escribe la palabra reservada 'si', seguido entre parentesis se escribe una condición, dicha condición puede ser: mayor que (>), menor que (<), igual a (==), diferente de (!=), mayor o igual (>=), y tambien menor o igual (<=).
Segido se escribirá dentro de bloques '{}' las sentencias a ejecutar. En caso de no complirse la condición, se puede colocar luego de cerrar los bloques la palabra reservada 'no' y seguido escribir en bloques un código que se ejecutará.
Sentencia para:
Primero se escribe la palabra reservada 'para', seguido entres parentesis se debe iniciar una variable, escribir una condición, y por último una asignación al valor de la variable para cuando se ejecute el ciclo, luego dentro de un bloque '{}' se debe escribir las sentencias a ejecutar por el ciclo.
Sentencia mientras:
Se escribe la palabra reservada 'mientras', luego dentro de parentesis se escribe una condición, y al final en un bloque se colocan las sentencias a ejecutar.
Sentencia haz:
Funciona como el mientras, solo que en este caso se escribe primero la palabra reservada 'haz', seguido se escribe un bloque '{}' con instrucciones, y al final del bloque se escribe la palabra mientras seguido de un parentesis con la condición y al final punto y coma. Esta a diferencia del mientras ejecutará por lo menos una vez el bloque de código.

A tener en cuenta:
Se puede anidar código, ejemplo, un if dentro de un for, un for dentro de un mientras y así con todas las sentencias.
